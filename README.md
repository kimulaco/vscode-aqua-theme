# vscode-aqua-theme

Color theme converted from [Aqua Theme](https://github.com/cafarm/aqua-theme) for Visual Studio Code.

- Light Theme
    - `Espresso Aqua`
    - `Tomorrow Night Aqua`
- Dark Theme
    - `Monokai Aqua`
    - `Tomorrow Aqua`
